# Media Assets

This directory stores bundled visual assets used by PetBuddy2.

## Asset Types

- `*.tga` - Frame art, borders, buttons, status bar textures, and icons.
- `*.png` - Alternate logo asset.
- `*.ttf` - Bundled font file.
- `sources.txt` - Attribution notes for third-party media sources.

## Usage Notes

- Keep asset filenames lower case to match the current project convention.
- Update Lua/XML references whenever assets are renamed.
- Only ship assets that are actually used by the addon or documented as retained release assets.
