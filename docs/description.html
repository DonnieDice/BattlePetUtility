# RGX | Battle Pet Utility! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## Overview **RGX | Battle Pet Utility!** is a battle pet control HUD for World of Warcraft Retail Midnight that keeps your team status, pet utility actions, and loadouts in one compact, draggable frame. Built to stay usable during normal pet content and powered by RGX-Framework. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) **The Kiwi Says:** "Your pets deserve a better HUD! Bwwiiiee." *** ## Features Team HUD - Shows active team slots with health and XP bars at a glance. Auto Team Save - Pet teams automatically save and restore on login, so your last active team is always ready. Drag and Drop - Move pets between battle slots quickly and intuitively. Ability Controls - Swap active pet abilities directly from the frame during battle. Utility Quick Buttons - Heal, bandage, currencies, stones, rewards, and pet consumables - all one click away. Native Loadouts - BPU includes account-wide native loadouts for save, rename, overwrite, restore, and delete support. Rematch Integration - When Rematch is installed, BPU can work with the Rematch team save/load flow. Zone Pet Tracker - Built-in BPU zone progress with completion bars, missing pets, and collected-pet visibility. Covers modern and legacy pet zones. PetTracker Compat - When PetTracker is installed, BPU plays nicely with its live zone data and pet journal behavior. Minimap Icon - Standalone draggable minimap button using the BPU logo. Left-click toggles the frame, right-click opens options, and the icon can be repositioned around the minimap edge. Auto Heal - Stable-master automation support for hands-free healing between battles. Auto Resummon - Companion resummon with trigger handling for common state transitions. Charms Display - Aggregates pet charm item and currency counts in one place. UI Controls - Scale, font, texture, visibility options, frame opacity, and a minimize toggle - fully customizable. **Note:** Enhanced when Rematch or BattlePetBreedID is installed - works without them too. *** ## Slash Commands The addon can be managed using the following slash commands:
/BattlePetUtility # Toggle the Battle Pet Utility! frame
/BattlePetUtility help # Show command help
/BattlePetUtility welcome # Toggle login welcome message
/BattlePetUtility version # Print installed addon version
/bpu # Alias for /BattlePetUtility
