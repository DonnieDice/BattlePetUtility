# 🐾 RGX | PetBuddy2 - Battle Pet HUD & Utility ![PetBuddy2 Logo](https://github.com/donniedice/PetBuddy2/blob/main/images/icon.tga?raw=true) ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | PetBuddy2** is a battle pet control HUD for World of Warcraft Retail Midnight that keeps your team status, pet utility actions, and loadouts in one compact, draggable frame. Built to stay usable during normal pet content without requiring any external embedded libraries. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "Your pets deserve a better HUD! Bwwiiiee." *** ## ✨ Features 🐾 Team HUD - Shows active team slots with health and XP bars at a glance! 🔀 Drag and Drop - Move pets between battle slots quickly and intuitively. ⚔️ Ability Controls - Swap active pet abilities directly from the frame during battle. 🧰 Utility Quick Buttons - Heal, bandage, currencies, stones, rewards, and pet consumables — all one click away! 💾 Native Loadouts - Save, rename, overwrite, restore, and delete loadouts without needing Rematch. 🤝 Optional Rematch Helper - Shift-click Save to open Rematch save flow when Rematch is installed. 🏥 Auto Heal - Stable-master automation support for hands-free healing between battles. 🔄 Auto Resummon - Companion resummon with trigger handling for common state transitions. 💎 Charms Display - Aggregates pet charm item and currency counts in one place. 🎨 UI Controls - Scale, font, texture, visibility options, and a minimize toggle — fully customizable! **Note:** Enhanced when Rematch or BattlePetBreedID is installed — works great without them too! *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/petbuddy # Toggle the PetBuddy2 frame
/petbuddy help # Show command help
/petbuddy welcome # Toggle login welcome message
/petbuddy version # Print installed addon version
/pb # Alias for /petbuddy
/bpb # Alias for /petbuddy
*** ## 🎮 Compatibility Retail Midnight | ✅ Fully Supported (Interface 120000, 120001) **Optional Addons:** Rematch, BattlePetBreedID, LibDataBroker-1.1 (no required libraries) *** ## 📥 Installation 1. **Download** RGX | PetBuddy2 from your preferred platform: * [CurseForge](https://www.curseforge.com/wow/addons/petbuddy2) * [GitHub](https://github.com/donniedice/PetBuddy2) * [Wago.io](https://addons.wago.io/addons/petbuddy2) * [WoWInterface](https://www.wowinterface.com/downloads/info-PetBuddy2.html) 2. **Extract** to your WoW `Interface/AddOns` directory: * **Retail WoW:** World of Warcraft/retail/Interface/AddOns 3. **Restart** WoW and enable the addon in the AddOns menu. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "Your pets are ready for battle!
